Datapack to Fabric Mod Converter

A standalone desktop application developed with Python and CustomTkinter that streamlines the process of converting Minecraft datapacks into fully functional Fabric mods. The tool features an intuitive graphical user interface (GUI) that allows users to simply select their datapack, and the application handles the rest. It automatically generates the necessary metadata, compiles the code, and packages everything into a distributable .jar file, significantly reducing the complexity and time required for mod creation.
